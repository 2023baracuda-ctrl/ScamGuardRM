Politica de confidențialitate

În vigoare din 30 iunie 2026. Versiunea 1.1

ScamGuard este o aplicație Android gratuită care ajută la prevenirea fraudelor telefonice. Această politică explică ce date prelucrează aplicația, unde sunt păstrate și ce se transmite în afara dispozitivului. Pe scurt: conținutul SMS-urilor dvs. nu părăsește niciodată dispozitivul.

1. Cine suntem

ScamGuard este un proiect personal non-comercial, dezvoltat de o persoană fizică din Chișinău, Republica Moldova. Contact: scamguardrm@gmail.com.

În sensul Regulamentului UE 2016/679 (GDPR) și al Legii Republicii Moldova nr. 133/2011 privind protecția datelor cu caracter personal, operatorul de date în cadrul acestei aplicații este dezvoltatorul.

2. Ce date prelucrează aplicația

Aplicația lucrează cu trei categorii de date, iar toate rămân pe dispozitivul dvs.:

  • Conținutul SMS-urilor primite — analizat local pentru identificarea codurilor unice și a legăturii cu un apel activ. Nu se transmite în afara dispozitivului.
  • Statusul apelului telefonic — aplicația știe dacă este un apel activ chiar acum și ce număr a sunat ultima dată. Aceste date sunt folosite doar în momentul funcționării și nu se păstrează.
  • Istoricul detecțiilor — lista cazurilor când aplicația a afișat un avertisment. Se păstrează doar în memoria dispozitivului, poate fi șters din Setări.

3. Ce se transmite pe server

Aplicația face o singură cerere de rețea de ieșire:

  • Confirmare anonimă a acceptării termenilor. La prima rulare, când acceptați această Politică și Acordul utilizatorului, pe server se trimite o înregistrare anonimă: versiunea termenilor, data acceptării și hash-ul SHA-256 al identificatorului de sistem al dispozitivului Android. Din acest hash este imposibil de recuperat numele, numărul, email-ul sau alte informații personale ale dvs. — este doar un șir de 64 de caractere.

Această înregistrare este stocată pe serverele Cloudflare (KV Storage) și folosită exclusiv ca dovadă juridică a obținerii consimțământului. Termenul de păstrare — nelimitat, deoarece ștergerea ar lipsi dovada de bază. Aceasta corespunde excepției din articolul 17(3)(e) GDPR.

4. Ce NU face aplicația

  • Nu transmite conținutul SMS-urilor nicăieri
  • Nu stochează date în cloud
  • Nu folosește analytics (Google Analytics, Firebase Analytics etc.)
  • Nu afișează reclame
  • Nu transmite date către terți
  • Nu folosește cookies sau alte tehnologii de urmărire
  • Nu solicită acces la lista de contacte, fotografii, fișiere, locația GPS

5. Servicii terțe

Aplicația folosește Cloudflare Workers și Cloudflare KV pentru stocarea confirmărilor anonime ale acceptării termenilor. Cloudflare procesează datele pe teritoriul UE. Politica lor de confidențialitate este disponibilă la cloudflare.com/privacypolicy.

Aplicația este distribuită prin GitHub Releases și (planificat) Google Play. La descărcare se aplică politicile de confidențialitate ale GitHub sau Google.

6. Drepturile dvs.

Conform GDPR și Legii nr. 133/2011, aveți următoarele drepturi:

  • Dreptul de a ști ce date despre dvs. sunt prelucrate — acest document răspunde exhaustiv la această întrebare
  • Dreptul la ștergere — toate datele locale de pe dispozitivul dvs. pot fi șterse prin curățarea datelor aplicației din setările de sistem Android. Hash-ul anonim de pe server nu vă identifică personal și este păstrat ca dovadă juridică (excepția art. 17(3)(e) GDPR)
  • Dreptul de a obiecta — puteți înceta să folosiți aplicația oricând, pur și simplu dezinstalând-o
  • Dreptul de a depune o plângere — la Centrul Național pentru Protecția Datelor cu Caracter Personal al Moldovei (datepersonale.md) sau la autoritatea de supraveghere din țara dvs. de reședință

7. Copii

Aplicația este destinată persoanelor cu vârsta de peste 18 ani. Acceptând Acordul utilizatorului, confirmați că aveți cel puțin 18 ani.

8. Securitate

Deoarece datele principale sunt stocate local pe dispozitiv, securitatea depinde în primul rând de cât de bine este protejat dispozitivul însuși (cod PIN, criptare). Pentru partea de server folosim măsuri standard de protecție Cloudflare.

9. Modificări ale politicii

Dacă această politică se schimbă semnificativ, aplicația va afișa o notificare și va cere reacceptarea termenilor. Corectările minore sunt publicate pe această pagină cu actualizarea datei la începutul documentului.
